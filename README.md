
# Rokto-Data

Rokto-Data is an app for managing Blood Donors. It's suitable for Blood Bank Admins to quickly find donors from their donor dataabse.

## Features

- Donors List
- Search Donors
- Filter by Blood Group
- Data Save for Offline
- Add/Edit/Delete Donors
- Call Donors Locally from One Tap
- Save Emergency and Hospitals Phone Number
